❓ Hỏi đáp nhanh

Web vs App native — khác nhau không?

Không khác chút nào về khả năng. Mọi việc đều phải đi qua ADB (cầu nối Mac ↔ máy). Tool web này thực chất là 1 trang web local gọi ADB qua Python — kết quả y hệt như app native.

App native trên điện thoại không làm được debloat — Android cấm app thường gỡ system app.

"Tắt" và "Gỡ" khác nhau thế nào?

Tắt (disable): app vẫn còn trên máy, ẩn icon, không chạy ngầm. Bật lại 1-click. → Khuyến nghị dùng cái này.

Gỡ cho user 0 (uninstall): gỡ cho user hiện tại, file APK vẫn trong system. Khôi phục bằng pm install-existing. Sạch hơn nhưng factory reset thì app quay lại.

Tôi tắt nhầm app, máy lỗi?

Tool đã chặn 60+ package cốt lõi không cho tắt. Nếu máy lạ:

  1. Vào tab Tất cả app, lọc Đã tắt.
  2. Tìm app vừa tắt, bấm Bật.
  3. Nếu không nhớ tắt cái nào: tick tất cả → Bật đã chọn để khôi phục về trạng thái gốc.
  4. Cuối cùng: Cài đặt → Hệ thống → Đặt lại → Đặt lại thiết bị (mất data).
Đổi font hệ thống được không?

KHÔNG, nếu không có root. Sony khoá custom font ở firmware level. Cách workaround:

  • Đổi font tên app: cài Nova Launcher, Niagara Launcher, Lawnchair.
  • Đổi cỡ chữ toàn máy: Cài đặt → Hiển thị → Cỡ chữ.
  • Đổi font trong app: nhiều app (Word, Notion, Kindle) cho phép đổi font riêng.
Tại sao không bẻ được bootloader?

XQ-AS42 là bản nội địa Nhật. Sony chính sách: không cấp mã unlock cho thị trường Nhật. Snapdragon 865 có secure boot — không có exploit công khai nào bẻ được ở Android 10+. Ai bán "dịch vụ unlock" cho dòng này → 99% là lừa, gạch máy.

Sau khi tắt 1 đống app, nên làm gì?
  1. Vào tab Tối ưu → áp dụng "Tắt animation" + "Tắt Wi-Fi scan liên tục" + "Tắt AOD".
  2. Khởi động lại máy.
  3. Dùng máy 1-2 ngày bình thường.
  4. Nếu thiếu chức năng → bật lại app đó.
  5. OK rồi → quay lại tab Dọn sạch, chọn mức cao hơn.
