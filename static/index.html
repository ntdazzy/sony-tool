Hướng dẫn chi tiết

    Kiểm tra Bootloader

    Xem trạng thái lock + máy có đủ điều kiện unlock không

    Trạng thái bootloader

    Đọc trạng thái lock + verified boot state từ máy qua ADB.

    Về Sony unlock bootloader

    • Máy thị trường Nhật (XQ-AS42, SO-52A, SOG02, A002SO...): Sony KHÔNG cấp mã unlock. Không có exploit công khai. Bỏ ý định.
    • Máy quốc tế (XQ-AS52, XQ-AS62, XQ-BC42...): có thể đủ điều kiện. Kiểm tra IMEI ở Sony site.
    • Quy trình unlock chính thức:
      1. Dial *#06# để lấy IMEI
      2. Vào Sony Open Devices
      3. Nhập IMEI → nếu eligible → Sony gửi mã unlock
      4. Trên máy: Settings → Developer Options → OEM unlocking ON
      5. Reboot vào fastboot, chạy: fastboot oem unlock 0x<mã>
    • Cảnh báo: unlock = factory reset + mất bảo hành + mất DRM keys (camera quality giảm, không xem HDR Netflix).

    Hỏi đáp

    Các câu hỏi thường gặp

    Cơ chế tối ưu hoạt động thế nào?

    Tool dùng 2 lớp tối ưu:

    Lớp 1 — Tắt/gỡ app rác (qua pm disable-user hoặc pm uninstall --user 0):

    • App không khởi động được, service/receiver không chạy ngầm
    • Cron jobs / scheduled task của app không trigger
    • → Bớt 30-40% RAM dùng ngay khi máy bật, bớt CPU/pin cho background

    Lớp 2 — Tinh chỉnh system settings (qua settings put + device_config put):

    • Animation off → GPU không phải vẽ transitions → cảm giác phản hồi tức thì
    • max_phantom_processes / max_cached_processes → ít app cached trong RAM → app mới mở nhanh hơn
    • wifi_scan_always_enabled = 0 → radio Wi-Fi không scan ngầm → pin tiết kiệm
    • doze_always_on = 0 → màn hình tắt hoàn toàn lúc idle → pin tiết kiệm
    • limit_ad_tracking = 1 → app không track bạn → giảm data + privacy
    • captive_portal_mode = 0 → máy không ping Google mỗi lần connect Wi-Fi

    Mỗi tweak nhỏ nhưng cộng dồn: ~20% pin tiết kiệm standby, snappier feel rõ rệt.

    Tool có chạy được trên Sony khác ngoài XQ-AS42?

    Có. Bloat list curate theo XQ-AS42 nhưng:

    • ~80% package là universal Sony bloat (com.sonymobile.*, com.sonyericsson.*, Facebook system apps) — chạy được trên Xperia 1, 5, 10 series
    • Tool tự lọc "chỉ hiện app đang có trên máy" → package nào không tồn tại sẽ ẩn
    • Settings tweaks dùng AOSP keys → chạy được trên mọi Android (không Sony-specific)
    • Nếu máy bạn có bloat lạ chưa trong list → vào Backup → Xuất chi tiết gửi dev cập nhật

    Đã verify trên: XQ-AS42, SO-52A, Xperia 1 III. Lý thuyết hoạt động với Xperia 5/10 series tương tự.

    Badge "Đã áp dụng" / "Mặc định" / "1 phần" nghĩa là gì?

    Khi mở tab Tối ưu, tool đọc value hiện tại của mỗi setting trên máy (qua settings get / device_config get), so sánh với value mà preset sẽ set:

    • Đã áp dụng (xanh): tất cả setting trong preset đã match → preset đang active
    • Mặc định (xám): tất cả setting là giá trị mặc định → preset chưa áp dụng
    • 1 phần (vàng): một phần setting đã thay đổi (vd: preset có 3 step, đã apply 1) → có thể do bạn từng đổi tay 1 setting
    • — (chưa rõ): preset dùng shell command không đọc back được (vd: dumpsys deviceidle)

    Hover badge để xem giá trị hiện tại vs giá trị target.

    Web vs App native — khác nhau không?

    Không khác về khả năng. Mọi việc đều phải đi qua ADB (cầu nối Mac/PC ↔ máy). Tool web này thực chất là trang web local gọi ADB qua Python — kết quả y hệt app native.

    App native trên điện thoại không làm được debloat — Android cấm app thường gỡ system app.

    "Tắt" và "Gỡ" khác nhau thế nào?

    Tắt (disable): app vẫn còn trên máy, ẩn icon, không chạy ngầm. Bật lại 1 click.

    Gỡ cho user 0 (uninstall): gỡ cho user hiện tại, file APK vẫn trong system. Khôi phục bằng pm install-existing. Sạch hơn nhưng factory reset thì app quay lại.

    Tôi tắt nhầm app, máy lỗi?

    Tool đã chặn 76 package cốt lõi không cho tắt. Nếu máy lạ:

    1. Vào tab Tất cả app, lọc Đã tắt.
    2. Tìm app vừa tắt, bấm Bật.
    3. Cuối cùng: Cài đặt → Hệ thống → Đặt lại → Đặt lại thiết bị.
    Đổi font hệ thống được không?

    KHÔNG nếu không root. Workaround:

    • Cài Nova/Niagara Launcher (đổi font tên app)
    • Đổi cỡ chữ: Cài đặt → Hiển thị → Cỡ chữ
    • Đổi font trong app: Word, Notion, Kindle cho phép
    Tại sao không bẻ được bootloader?

    XQ-AS42 là bản nội địa Nhật. Sony chính sách: không cấp mã unlock cho thị trường Nhật. Snapdragon 865 có secure boot — không có exploit công khai cho Android 10+. Ai bán dịch vụ unlock cho dòng này: 99% là lừa.

    Sau khi tắt một loạt app, nên làm gì?
    1. Vào Tối ưu → áp dụng "Tắt animation" + "Tắt Wi-Fi scan" + "Tắt AOD"
    2. Khởi động lại máy
    3. Dùng 1-2 ngày bình thường
    4. Nếu thiếu chức năng → bật lại app đó
